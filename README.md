## 简介

ZakoBili 是一个专为 [哔哩哔哩](https://mall.bilibili.com) 设计的票务选购工具喵~ (｡◕‿◕｡)

通过调用手机端接口，帮助大家更方便快捷地选购心仪的票务，再也不用担心手速不够快啦！✨
